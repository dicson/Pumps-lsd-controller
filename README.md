# Pumps-lsd-controller

Система автоматического управления насосными станциями с графическим интерфейсом пользователя. Проект представляет собой комплекс из главного контроллера с сенсорным экраном и удаленных модулей управления реле и мониторинга.

## 🚀 Основные возможности

- **Графический интерфейс (GUI):** Современный интерфейс на базе **LVGL 9.x**, разработанный в EEZ Studio.
- **Беспроводное управление:** Использование протокола **ESP-NOW** для связи между главным контроллером и периферийными модулями с минимальной задержкой.
- **Гибкая логика полива:** Поддержка многозонного управления с разделением фаз «грязной воды» (`dw_time`) и «чистой воды» (`cw_time`).
- **Мониторинг в реальном времени:** Отслеживание тока потребления насосов через удаленные сенсоры для обнаружения сбоев.
- **OTA-обновления:** Дистанционное обновление прошивки всех модулей через веб-интерфейс (**ElegantOTA**).
- **Энергосбережение:** Автоматическое диммирование подсветки экрана при отсутствии активности.

## 🛠 Технический стек

- **Микроконтроллеры:** ESP32-S3 (главный контроллер), ESP32 (модули реле и сенсоры).
- **Дисплей:** 7-дюймовый RGB LCD (800x480) с тачскрином GT911.
- **Библиотеки:**
  - `lvgl` (v9.5) — графический интерфейс.
  - `Arduino_GFX` — драйвер дисплея.
  - `TAMC_GT911` — драйвер тачскрина.
  - `ElegantOTA` — обновление ПО по воздуху.
- **Среда разработки:** PlatformIO (Arduino framework).

## 📐 Архитектура системы

Система состоит из четырех основных компонентов:

1. **Главный контроллер (Pumps-lsd-controller):**
   - Управляет всей логикой системы.
   - Отображает текущее состояние и позволяет менять настройки.
   - Отправляет команды модулям реле по ESP-NOW.

2. **Модуль управления реле ([Relay Module 1](https://github.com/dicson/relay-module-1)):**
   - Принимает команды от главного контроллера.
   - Управляет до 16 реле через сдвиговые регистры 74HC595.
   - Обеспечивает высокую плотность коммутации при малом количестве пинов ESP32.

3. **Модуль датчика тока ([c_sensor](https://github.com/dicson/c_sensor)):**
   - Измеряет ток потребления насоса.
   - Передает данные обратно на главный контроллер по ESP-NOW для контроля работы оборудования.

4. **Дистанционный пульт ([pult](https://github.com/dicson/pult)):**
   - Обеспечивает удаленное управление системой без необходимости взаимодействия с основным экраном.

## 📦 Сборка и установка

### Требования
- [PlatformIO](https://platformio.org/)

### Сборка
1. Откройте проект в VS Code с установленным расширением PlatformIO.
2. Выполните команду сборки:
   ```bash
   pio run
   ```
3. Загрузите прошивку в устройство:
   ```bash
   pio run --target upload
   ```
4. Откройте монитор порта для отладки:
   ```bash
   pio device monitor
   ```

## 📂 Структура проекта
- `src/ui/` — исходный код интерфейса (генерируется EEZ Studio).
- `src/auto_pumps.cpp` — основная логика управления насосами и таймерами.
- `src/enow.cpp` — реализация обмена данными по ESP-NOW.
- `src/display.cpp` — инициализация и управление дисплеем/подсветкой.
- `include/constants.h` — глобальные константы и настройки системы.
