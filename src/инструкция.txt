Инструкция пользователя контроллером Pumps-LSD

  Данный контроллер предназначен для управления автоматизированной
  системой полива, состоящей из 30 зон. Система поддерживает двухфазный
  полив (грязная и чистая вода) и дистанционное управление
  исполнительными механизмами по протоколу ESP-NOW.

  1. Основной экран (Управление)
  Основной экран предназначен для запуска программы и мониторинга
  текущего состояния.

   * Кнопка СТАРТ: Запускает цикл полива для всех настроенных зон по
     очереди.
   * Кнопка СТОП: Немедленно прекращает выполнение программы, выключает
     насос и все активные зоны.
   * Индикаторы состояния:
       * Иконка Насоса: Появляется, когда основной насос включен.
       * Иконка Трубы/Осмоса: Показывает, какая вода подается в данный
         момент — «грязная» (техническая) или «чистая» (фильтрованная).
       * Шкала прогресса: Отображает общее время выполнения программы и
         текущий этап.
       * Панель зон: 30 индикаторов (столбиков), которые показывают
         прогресс полива в каждой конкретной зоне. Синий цвет столбика
         означает фазу грязной воды.

  2. Настройка зон (Вкладка «Зоны»)
  На этой вкладке вы задаете время работы для каждой из 30 зон.

   * Настройка времени: Каждая зона имеет два параметра:
       * D (Dirty): Время работы с «грязной» водой (в минутах).
       * C (Clear): Время работы с «чистой» водой (в минутах).
   * Как изменить: Нажмите на значение времени, введите нужное число на
     появившейся клавиатуре и нажмите «Enter» (галочка).
   * Визуализация: Активные зоны (где время больше 0) подсвечиваются
     ярко. Неиспользуемые зоны выглядят полупрозрачными.
   * Выборочный запуск: Вы можете нажать на номер зоны на вкладке
     ручного выбора, чтобы запустить полив только для этой зоны.

  3. Настройки системы (Вкладка «Настройки»)
  Здесь собраны параметры оборудования и общие коэффициенты.

   * Яркость: Слайдер для регулировки подсветки экрана.
   * Время засыпания: Время в секундах, через которое экран погаснет
     при бездействии.
   * Пауза: Время ожидания (в секундах) между переключениями зон
     (необходимо для выравнивания давления в системе).
   * Коэффициент Кгв (k_dw_time): Глобальный множитель для времени
     «грязной» воды. Позволяет оперативно увеличить или уменьшить полив
     технической водой для всех зон сразу (от 50% до 200%).
   * Дистанционное управление (Пульт): 
       * Включение/выключение поддержки внешнего пульта.
       * Выбор протокола связи: ESP-NOW или LoRa.
   * Датчик насоса: Если активирован, система будет отслеживать
     состояние насоса и выдавать предупреждение при сбоях.
   * Обновление: Кнопка перехода в режим беспроводного обновления
     прошивки (OTA).

  4. Логика работы цикла
  При нажатии кнопки «СТАРТ» контроллер выполняет следующие действия:
   1. Проверяет все зоны по порядку с 1 по 30.
   2. Если для зоны задано время полива:
       * Включается фаза Dirty (грязная вода). Время работы = Заданное
         время * Коэффициент Кгв.
       * Включается фаза Clear (чистая вода).
       * Выполняется Пауза перед переходом к следующей зоне.
   3. По завершении всех зон насос автоматически выключается.

  5. Безопасность и ошибки
   * Ошибка связи: Если контроллер потеряет связь с релейным модулем по
     ESP-NOW, система может перейти в состояние ошибки. Убедитесь, что
     исполнительные блоки включены и находятся в зоне доступа.
   * Датчик сухого хода: Если система обнаружит, что насос работает
     «всухую» (при наличии соответствующего датчика), программа будет
     остановлена для предотвращения поломки оборудования.
